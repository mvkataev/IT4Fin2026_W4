## Трансформация биржевых данных: Мини-проект IV.

В файле W4_16_Катаев.R содержится код для работы со свечами и стаканом заявок.

Все задания выполнены в полном объеме, в том числе проведен рефакторинг функций в задании 3. Для использования функций в коде необходимо перейти на строчку 603. Предлагается задать несколько параметров:

— day (стартовый день в формате YYYY-MM-DD или YYYYMMDD); 

— sec (тикер акции, торгующейся на МосБирже); 

— freq (параметр частоты, который использовался при рефакторинге в задании 3).

В задании 2 для freq указывается количество секунд (по условию), а в задании 1 пользователь может вводить произвольный интервал (по условию, в том числе в виде "5 mins", "3 часа" или "1 d").
